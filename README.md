| 👨‍⚕️ Doctor Management | Doctor profiles, scheduling & availability |
| 📅 Appointment Booking | Book, update & track OPD appointments |
| 🗂️ Medical Records | Store and retrieve patient history |
| 🔐 JWT Auth | Secure API access with token-based authentication |
